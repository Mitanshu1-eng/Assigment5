Task 1 
This program stores some student names and their marks in a dictionary.
The user types a name, and the program shows the marks of that student.
If the name is not in the dictionary, it shows “Student not found.”

Task 2 

This program makes a list of numbers from 1 to 10.
It takes the first five numbers from the list, then reverses them.
Finally, it prints the original list, the first five numbers, and the reversed list.
